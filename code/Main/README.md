> [!CAUTION]<br>
> The underlying work for this repository is currently under review. Until the work is accepted for publication, all content should be considered as preliminary draft and may contain errors.


This repository relies on an in-house life cycle assessment software `lce.jl`. Current the software is not publicly available as it is:

1. still under development
2. hard to use for external users
3. not documented

We therefore invite readers to base their replication efforts using the [tutorial materials](../../Tutorial/) and both [the inventory](./modules/inventories.jl) and [the optimisation code](./modules/optimisation.jl).
